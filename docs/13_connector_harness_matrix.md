## 1. 目的

本文件用于把外部接口、线束、电流等级、屏蔽和防呆要求集中起来。

当前状态：

- 连接器系列未指定。
- 线束长度未指定。
- 防水等级未指定。
- 电流等级未冻结。

因此本文只定义接口需求，不指定最终料号。

## 3. 大电流连接器要求

适用：

- 电池输入。
- 轮毂三相输出。
- 转向三相输出，若电流较大。
- 仓门三相输出，若电流较大。

必须确认：

- 最大持续电流。
- 最大峰值电流。
- 线束长度。
- 线径。
- 是否频繁插拔。
- 防水等级。
- 是否需要锁扣。
- 是否需要防反插。

设计约束：

- 电池和三相输出不应使用容易误插的同型小信号连接器。
- 三相输出连接器要考虑爬电/电气间隙、发热、振动松脱。
- 若电机线有屏蔽层，屏蔽接地策略需在结构层面确认。

## 4. 轮毂线束

每个轮毂建议至少两组线束：

1. 三相功率线。
2. Hall/温度/编码器信号线。

### 4.1 三相功率线

| Pin | 信号 | 线径 | 备注 |
|---:|---|---|---|
| 1 | `U` | TBD_BY_USER | 大电流 |
| 2 | `V` | TBD_BY_USER | 大电流 |
| 3 | `W` | TBD_BY_USER | 大电流 |
| 4 | `SHIELD/PE` | TBD | 若使用屏蔽电机线 |

### 4.2 Hall 信号线

| Pin | 信号 | 方向 | 保护 |
|---:|---|---|---|
| 1 | `5V_HALL` | 板 -> 电机 | 限流/滤波 |
| 2 | `GND` | - | 信号地 |
| 3 | `HALL_A` | 电机 -> G4 | ESD/滤波/电平保护 |
| 4 | `HALL_B` | 电机 -> G4 | ESD/滤波/电平保护 |
| 5 | `HALL_C` | 电机 -> G4 | ESD/滤波/电平保护 |
| 6 | `AUX_TEMP_SPEED` | 电机 -> G4 | 类型 TBD_BY_USER |
| 7 | `SHIELD_GND` | - | 可选 |

## 5. 转向线束

每个转向机构建议：

- 三相电机线。
- 输出轴编码器线。
- 机械限位线。

### 5.1 输出轴编码器线

| Pin | 信号 | 方向 | 备注 |
|---:|---|---|---|
| 1 | `3V3_ENC` | 板 -> 编码器 | 滤波/限流 |
| 2 | `GND` | - | 信号地 |
| 3 | `SPI_SCK` | G4 -> 编码器 | 串阻 |
| 4 | `SPI_MISO` | 编码器 -> G4 | 串阻 |
| 5 | `SPI_MOSI` | G4 -> 编码器 | 可选 |
| 6 | `SPI_CS` | G4 -> 编码器 | 独立 CS |
| 7 | `DRDY_ERR` | 编码器 -> G4 | 可选 |
| 8 | `SHIELD_GND` | - | 可选 |

### 5.2 转向限位线

待用户确认机械形式后定。

建议至少预留：

| 信号 | 说明 |
|---|---|
| `STF_LIMIT_L` | 前转向一侧限位 |
| `STF_LIMIT_R` | 前转向另一侧限位 |
| `STR_LIMIT_L` | 后转向一侧限位 |
| `STR_LIMIT_R` | 后转向另一侧限位 |

待确认：

- 限位开关是常开还是常闭。
- 是否共地。
- 是否需要独立供电。
- 线束是否长，是否需要更强抗干扰。

## 6. UART 传感器线束

### 6.1 每路 UART 接口

| Pin | 信号 | 说明 |
|---:|---|---|
| 1 | `VOUT` | 3.3 V/5 V 跳线选择 |
| 2 | `GND` | - |
| 3 | `TX_TO_SENSOR` | 板 -> 外设 |
| 4 | `RX_FROM_SENSOR` | 外设 -> 板 |
| 5 | `SHIELD_GND` | 可选 |

### 6.2 电平要求

- MCU/扩展芯片侧固定 3.3 V。
- 外设侧可选择 3.3 V 或 5 V。
- TX/RX 必须经过电平转换。
- 不能让 5 V UART 直接进入 MCU。

### 6.3 待确认

- 超声波模块型号。
- 每路电流。
- 波特率。
- 线束长度。
- 是否需要单独开关每路电源。

## 7. USB Type-C/Ethernet 机械问题

已冻结：

- USB 接口使用 Type-C。
- Ethernet 必须有网口，优先板载 RJ45。

仍需确认：

- 外壳是否金属。
- RJ45 是否需要带灯。
- RJ45 使用带磁性器件版本还是外置网络变压器。
- Type-C 是否需要从 VBUS 给少量调试电路供电。
- Type-C 和 RJ45 是否需要对外露出到外壳面板。
- 是否需要面板固定。

## 8. 连接器选型规则

### 8.1 不应混用

- 大电流电池输入和小信号接口不使用相似外形。
- 电机三相和 Hall 信号不使用容易误插的同型连接器。
- 24 V、5 V、3.3 V 外设接口需要清楚标识，避免插错。

### 8.2 建议

- 电池输入和电机三相使用带锁扣、足够电流、明确极性的连接器。
- 信号线使用带锁扣连接器。
- 车体外部接口优先考虑防水连接器。
- 调试接口可以用标准 2.54/1.27 排针，但量产时应考虑保护。

## 9. 丝印要求

每个外部连接器旁应标：

- 连接器编号。
- 关键针脚方向。
- 电压。
- 危险电源标识，若连接 VMOT/VBAT。
- 信号名称缩写。

大电流连接器旁建议标：

- `VBAT+`
- `VBAT-`
- `U/V/W`
- 警示符号。

## 10. 需要用户确认

1. 是否已有指定连接器系列？
2. 板子是否需要防水连接器？
3. 每个电机线束长度大概多少？
4. 电池输入线径和接插件目标是什么？
5. 三相电机线是否有屏蔽层？
6. USB Type-C 和 Ethernet 网口是否需要对外露出到外壳面板？
7. 调试口是否允许量产板保留？
