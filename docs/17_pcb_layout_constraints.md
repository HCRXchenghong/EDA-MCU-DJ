# PCB 布局布线约束初版

## 1. 状态

当前尚未确定：

- 板框。
- 层数。
- 铜厚。
- 最大电流。
- 散热方式。
- 连接器位置。

因此本文件给出布局布线原则和约束项，不给最终线宽和间距数值。

## 2. 分区原则

### 2.1 功能分区

| 分区 | 位置建议 | 关键要求 |
|---|---|---|
| 电源输入/安全区 | 靠近电池连接器 | 保险、反接、预充、接触器、TVS、大电流最短 |
| 前轮功率区 | 板边 | 三相输出靠边，本地母线电容靠近 MOS |
| 后轮功率区 | 板边 | 与前轮类似，避免与低压区交叉 |
| 转向功率区 | 板边 | 编码器信号要远离三相线 |
| 仓门功率区 | 靠近仓门连接器 | 根据电流决定远离 H7 程度 |
| 低压主控区 | 远离功率区 | H7、时钟、USB Type-C、ETH、IMU |
| 无线区 | 板边/角落 | ESP32 天线净空、禁铜、远离 MOS |
| 传感器接口区 | 靠近外部连接器 | UART/RS485/超声波，远离三相 |
| 调试/测试区 | 易接触 | SWD、测试点、状态 LED |

### 2.2 不允许的布局

- ESP32 天线下方铺铜或靠近大电流母线。
- 电流采样线穿过开关节点区域。
- Hall/编码器/UART 线穿过三相桥和母线电容之间。
- H7 时钟晶振靠近 MOS 开关节点。
- USB Type-C/Ethernet 差分线绕过功率区。
- 采样电阻 Kelvin 线和功率电流共用路径。

## 3. 层叠建议

具体层数由用户确认。

### 3.1 如果 4 层

推荐思路：

| 层 | 用途 |
|---|---|
| L1 | 元件/关键信号/局部功率 |
| L2 | 连续地平面，尽量完整 |
| L3 | 电源平面/低速信号 |
| L4 | 功率/信号/散热铜 |

风险：

- 大电流 4 层可能仍然紧张。
- 多路三相桥和低压高速通信共板，布局压力大。

### 3.2 如果 6 层

推荐思路：

| 层 | 用途 |
|---|---|
| L1 | 元件/短关键信号 |
| L2 | GND 平面 |
| L3 | 电源/功率局部 |
| L4 | 信号/电源 |
| L5 | GND/屏蔽参考 |
| L6 | 功率/散热/低速 |

优点：

- 更容易保证信号参考平面。
- 更容易隔离低压和功率噪声。
- CAN/USB Type-C/ETH/编码器信号质量更容易控制。

## 4. 大电流路径

需要用户给出最大电流后才能定线宽。

必须约束：

- 电池输入到主母线最短。
- 主母线到每个三相桥路径短且宽。
- 每个三相桥本地母线电容尽量靠近高低侧 MOS。
- 高 di/dt 回路面积最小。
- 三相输出连接器靠近对应 MOS 桥。
- 大电流过孔数量按电流和温升校核。
- 功率铜皮避免切割低压参考地。

## 5. 开关节点约束

开关节点包括：

- MOS 半桥中点。
- 三相 U/V/W。
- Bootstrap 相关高速节点。
- Gate 驱动回路。

约束：

- 开关节点铜皮面积不要无谓扩大。
- 远离电流采样、编码器、Hall、晶振、USB Type-C、Ethernet。
- Gate 电阻靠近 MOS Gate。
- Gate Driver 靠近 MOS。
- 高低侧驱动回路短。

## 6. 电流采样布局

三低边采样要求：

- 采样电阻靠近低侧 MOS Source。
- Kelvin 线从采样电阻内侧单独引出。
- 采样放大器靠近采样电阻。
- 差分采样线等长/靠近/远离开关节点。
- ADC RC 滤波靠近 ADC 或放大器输出，具体取决于放大器稳定性。
- 采样地回流不要承载功率电流。

三相采样要求：

- 传感器/采样器件靠近相线但远离高 dv/dt 干扰源。
- 注意共模范围。
- 注意隔离/磁传感器方向。

## 7. 通信布线约束

### 7.1 CAN FD

- CANH/CANL 成对走线。
- 外部 CAN 接口处放 TVS。
- 终端电阻位置靠近总线端。
- 外部 CAN 共模电感靠近连接器。
- 隔离 CAN 的隔离地和主地策略要清楚。

### 7.2 USB Type-C

- D+/D- 差分走线。
- ESD 靠近连接器。
- CC1/CC2 走线短，Rd 靠近连接器或 Type-C 保护电路。
- VBUS 检测/保护靠近连接器，明确不承载整板功率。
- 避免跨分割地。
- 避免靠近三相输出和 MOS。

### 7.3 Ethernet

- PHY 到磁性器件/RJ45 距离短。
- 差分对阻抗和长度按 PHY 要求。
- PHY 时钟远离功率区。
- RJ45 屏蔽接地策略待结构确认。

### 7.4 SPI 编码器

- SCK/MISO/MOSI/CS 串阻靠近驱动端。
- 编码器接口 ESD 靠近连接器。
- 远离三相线。
- 若线束长，需要降低 SPI 时钟或改接口方案，待编码器型号确认。

## 8. ESP32 布局约束

必须确认模块天线类型：

- PCB 天线。
- IPEX 外接天线。

PCB 天线版要求：

- 天线区域禁铜。
- 天线朝板边。
- 天线附近不要放金属件、大电流线、散热器。
- 若整机金属外壳包围，优先外接天线方案。

## 9. 测试点布局

必须易接触：

- 所有电源轨。
- H7 SWD。
- 3 个 G4 SWD。
- ESP32 下载/串口。
- CAN TX/RX 或 CANH/CANL。
- Driver Enable。
- Driver Fault。
- 电流采样输出。
- 母线电压采样。
- 急停状态。

功率测试点注意：

- 不要用小测试点承载大电流。
- 高压/大电流测试点需要安全间距和警示。

## 10. DRC/规则建议

待用户确认后设定：

- 最小线宽。
- 最小间距。
- 大电流线宽。
- 差分阻抗。
- 过孔尺寸。
- 铜厚。
- 最小丝印宽度。
- 禁布区。
- 高压/大电流间距。

不能在缺数据时确定：

- `PWR_VBAT_HIGH_CURRENT` 线宽。
- 三相线宽。
- 母线过孔数量。
- MOS 散热铜面积。
- 泄放电阻功率和位置。

## 11. PCB 审查重点

第一轮 PCB 审查必须看：

1. 电池输入到每个功率桥的回路。
2. 每个三相桥的母线电容位置。
3. 每个采样电阻 Kelvin 连接。
4. H7/ESP32 是否远离功率噪声。
5. Hall/编码器/UART 是否绕开功率区。
6. 急停链路是否物理独立且默认安全。
7. 外部接口 ESD 是否靠近连接器。
8. 调试口和测试点是否可接触。
