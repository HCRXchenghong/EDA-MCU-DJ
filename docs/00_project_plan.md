# 项目计划：外卖小车底盘控制大板

## 0. 设计输入

已有设计输入：

- 一块大 PCB，内部模块化分区。
- STM32H7 作为主控，负责上位机通信、传感器、调度、安全、仓门、电锁。
- 3 个 STM32G4 分别负责前轮双 FOC、后轮双 FOC、双转向位置 FOC。
- ESP32-S3 只作为 H7 的无线通信模块，不直接控制执行器。
- 4 个轮毂电机，单个标称 500 W，电压范围 24-48 V。
- 2 个转向无刷电机，转向角必须测输出轴。
- 1 个 5-24 V 仓门无刷编码电机，由 H7 直接控制。
- 1 个 5-24 V 电子锁，由 H7 控制。
- CAN FD、UART、RS485、USB Type-C、Ethernet 网口、ESP32、IMU、扩展 IO。
- 急停必须硬件关断，不能只进 MCU。

未明确但必须补齐的设计输入集中在 [04_open_questions.md](04_open_questions.md)。

## 2. 近期执行计划

### 2.1 当前正在做

- 建立硬件设计文档体系。
- 固化已确定架构。
- 识别不确定项。
- 形成原理图前的输入清单。

### 2.2 下一轮建议

在用户补齐 P0 参数后，进入以下动作：

1. 选定 H7 具体型号和封装。
2. 选定 G4 具体型号和封装。
3. 冻结电源输入范围、最大持续电流、峰值电流。
4. 计算每条电源轨电流预算。
5. 固化连接器型号和针脚定义。
6. 输出 H7/G4 详细引脚分配表。
7. 开始 EasyEDA 原理图页模板。

## 3. 项目风险矩阵

| 风险 | 等级 | 原因 | 当前处理方式 |
|---|---:|---|---|
| 多路 500 W 轮毂驱动功率回路 | P0 | 48 V 母线、大电流、开关噪声、散热压力高 | 先冻结峰值电流、铜厚、MOS、散热方式 |
| 双路 FOC 使用单个 G4 | P0 | PWM、ADC、比较器、DMA、编码器资源紧张 | 先做资源预算，再定具体 G4 型号/封装 |
| 急停硬件链路 | P0 | 不能依赖软件，必须能切断驱动使能和主接触器 | 独立安全链路设计，原理图专项审查 |
| 电流采样质量 | P0 | FOC 依赖低噪声同步采样 | 三低边或三相采样，不使用单母线采样作为主方案 |
| 24-48 V 宽输入电源 | P0 | 瞬态、反接、预充、刹车回灌、DCDC 压力 | 先做电源树和保护策略 |
| 转向真实轮角闭环 | P0 | 输出轴编码器机械安装影响控制安全 | 输出轴绝对编码器作为闭环主反馈 |
| H7 直控仓门 BLDC | P1 | H7 任务多，PWM/ADC/保护资源额外占用 | 引脚资源表单独预算，不够则改为独立小 MCU |
| 8 路超声波 UART | P1 | UART 扩展芯片、中断、缓冲、供电和电平转换复杂 | SPI-UART 扩展，接口限速，统一保护 |
| ESP32 与功率区共板 | P1 | 天线、EMI、地噪声 | 天线净空，远离 MOS 和母线 |
| 外部 5 V UART 输入 | P1 | 可能损坏 3.3 V MCU | 必须双电源电平转换，禁止直连 |

## 4. 通过标准

### 4.1 进入原理图的最低条件

- 板子层数、铜厚、板框尺寸、安装孔位置明确。
- 轮毂电机最大相电流或控制器最大峰值电流明确。
- 转向电机、仓门电机、电子锁参数明确。
- 主电池输入范围、保险、接触器、预充方案明确。
- H7/G4 具体型号和封装初选完成。
- 连接器类别和防呆/防水要求明确。

### 4.2 进入 PCB 的最低条件

- 原理图 ERC 通过，关键错误为 0。
- 所有电源轨有电流预算。
- 所有外部接口有 ESD/限流/电平保护策略。
- 所有 MCU 有 SWD、复位、BOOT、时钟、电源去耦。
- 所有电机驱动有过流、欠压/过压、过温、Fault、Enable。
- 急停硬件链路完整，不依赖软件。

### 4.3 进入打样的最低条件

- PCB DRC 通过，关键错误为 0。
- 功率回路宽度、过孔、铜厚、散热已按最大电流审查。
- 电流采样 Kelvin 连接和模拟地回流审查完成。
- CAN/UART/编码器/霍尔线没有穿越大电流噪声区。
- BOM 中所有器件有封装、替代料、耐压/温度/电流校核。

## 5. 不做的事

当前版本明确不做：

- 不做 7 个电机 MCU。
- 不做单 MCU 控全部底盘电机。
- ESP32 不直接控制电机、仓门、电锁、急停或安全逻辑。
- 不把急停只接进 MCU。
- 不把 5 V UART 直接接入 H7/G4/ESP32 的 3.3 V IO。
- 不用 TJA1050 做 CAN FD。
