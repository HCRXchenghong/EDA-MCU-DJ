# 设计审查关卡

## 1. G0：需求冻结审查

目标：确认可以进入器件和引脚资源选择。

必须完成：

- [ ] `16_user_input_form.md` 的 P0 参数基本填写。
- [ ] 电池最高/最低电压明确。
- [ ] 轮毂峰值电流明确。
- [ ] 板框、层数、铜厚方向明确。
- [ ] 急停复位策略明确。
- [ ] 主接触器策略明确。
- [ ] 转向编码器方案明确。

输出：

- 更新后的硬件规格书。
- 更新后的电源预算。
- 更新后的风险登记表。

## 2. G1：器件与资源审查

目标：确认核心器件可用于原理图。

必须完成：

- [ ] H7 型号和封装初选。
- [ ] G4 型号和封装初选。
- [ ] Gate Driver 方案初选。
- [ ] MOSFET 规格方向初选。
- [ ] DCDC 方案初选。
- [ ] CAN/RS485/UART 扩展器件初选。
- [ ] USB Type-C 连接器、CC/VBUS/ESD 方案初选。
- [ ] Ethernet PHY、RJ45/磁性器件、时钟和复位方案初选。
- [ ] 连接器系列初选。
- [ ] H7/G4 PinMux 初版。

通过标准：

- H7 外设不冲突。
- H7 必须保留 USB Type-C 与 Ethernet MAC/PHY 所需资源。
- G4 双 FOC 资源不冲突。
- 所有安全关键 IO 默认态清楚。
- 关键器件有采购路径或替代路线。

## 3. G2：原理图初版审查

目标：确认原理图结构正确。

必须完成：

- [ ] 14 个页面齐全。
- [ ] MCU 最小系统完整。
- [ ] 电源树完整。
- [ ] 通信物理层完整。
- [ ] USB Type-C 和 Ethernet 网口完整。
- [ ] 安全链路完整。
- [ ] 至少一个双 FOC 驱动模板完整。
- [ ] 连接器和测试点完整。
- [ ] ERC 初跑。

通过标准：

- 无关键 ERC 错误。
- 无未命名关键网络。
- 无 5 V 外部信号直入 MCU。
- USB Type-C 的 CC/VBUS/ESD 和 Ethernet 的 PHY/RJ45/ESD/屏蔽接地完整。
- 所有 Enable 默认禁用。
- 所有外部接口有保护策略。

## 4. G3：功率与安全专项审查

目标：确认不会因为功率/安全设计错误导致样板高概率损坏。

必须审查：

- [ ] MOS 耐压/电流/SOA/热。
- [ ] Gate Driver 电源和保护。
- [ ] 采样电阻功率。
- [ ] 采样放大器范围。
- [ ] 硬件过流阈值。
- [ ] 预充和接触器。
- [ ] 母线电容和 TVS。
- [ ] 刹车回灌/泄放。
- [ ] 急停硬件链路。
- [ ] 复位/默认态。

通过标准：

- P0 风险全部有处理方案。
- 所有安全关键链路可以在原理图中追踪。
- 上电默认禁用路径明确。

## 5. G4：PCB 布局审查

目标：确认 PCB 大结构可行。

必须审查：

- [ ] 功率区靠边。
- [ ] 主控远离功率区。
- [ ] ESP32 天线净空。
- [ ] 电池输入和母线回路。
- [ ] 每个三相桥本地母线电容。
- [ ] 采样 Kelvin。
- [ ] 通信差分线。
- [ ] 外部接口 ESD 靠近连接器。
- [ ] 测试点可接触。
- [ ] 散热路径。

通过标准：

- 功率回路和采样回路没有明显冲突。
- 大电流路径可按铜厚实现。
- 连接器位置与线束方向合理。

## 6. G5：PCB 完成审查

目标：确认可以输出打样资料。

必须完成：

- [ ] DRC 关键错误为 0。
- [ ] 网类规则完整。
- [ ] 丝印方向清楚。
- [ ] 装配方向清楚。
- [ ] BOM 无缺封装。
- [ ] Gerber 检查。
- [ ] 钻孔检查。
- [ ] Pick-and-place 检查。
- [ ] 未贴件 DNP 标注清楚。

通过标准：

- 生产文件可交给板厂/贴片厂。
- 风险接受项记录在案。

## 7. G6：样板上电审查

目标：确认样板上电步骤安全可控。

必须准备：

- [ ] 限流电源。
- [ ] 万用表。
- [ ] 示波器。
- [ ] CAN 工具。
- [ ] ST-Link/J-Link。
- [ ] 低压小电机或假负载。
- [ ] 急停按钮。
- [ ] 绝缘/散热/固定工装。

上电顺序：

1. 不接电池主功率，只测低压。
2. 不接电机，测 Gate/PWM。
3. 单通道限流测小电机。
4. 单轮毂空载。
5. 双轮毂。
6. 四轮毂。
7. 转向。
8. 仓门/电子锁。
9. 整车联调。

## 8. 任何时候必须停止的条件

- 不确定电源极性。
- 电流采样波形异常且原因未明。
- MOS/Gate Driver 过热。
- 急停不能可靠关断。
- CAN 超时后电机不进入安全状态。
- 母线电压异常升高。
- 5 V/3.3 V 轨异常掉压或过压。
