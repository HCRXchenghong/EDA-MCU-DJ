# CAN FD 协议骨架

## 1. 目标

定义 STM32H7 与各驱动区之间的内部 CAN FD 通信骨架。

当前状态：

- CAN ID、速率、报文周期、缩放系数未定。
- 本文件只定义协议框架，具体数值等待用户和固件策略确认。

## 2. 节点

| 节点 | 建议节点名 | 功能 |
|---|---|---|
| H7 主控 | `NODE_H7_MASTER` | 调度、命令、安全、诊断 |
| 前轮 G4 | `NODE_G4_FRONT` | 前左/前右轮驱动 |
| 后轮 G4 | `NODE_G4_REAR` | 后左/后右轮驱动 |
| 转向 G4 | `NODE_G4_STEER` | 前/后转向驱动 |

## 4. 安全原则

1. 所有驱动节点上电默认不使能。
2. 驱动节点必须收到 H7 周期心跳才允许保持使能。
3. CAN 超时后驱动节点进入安全状态。
4. H7 发送目标命令和使能命令应分离。
5. Fault 上报必须高优先级。
6. 急停硬件链路优先级高于 CAN 命令。

## 5. 报文分类

| 类型 | 方向 | 周期 | 说明 |
|---|---|---:|---|
| Heartbeat | H7 -> G4 | TBD | 主控心跳 |
| Node Status | G4 -> H7 | TBD | 节点状态、故障、温度、电压 |
| Motor Command | H7 -> G4 | TBD | 轮速/力矩/转向角命令 |
| Motor Feedback | G4 -> H7 | TBD | 速度、电流、角度、状态 |
| Fault Event | G4 -> H7 | 事件触发 | 故障锁存 |
| Config Write | H7 -> G4 | 低频 | 参数写入 |
| Config Read | H7 -> G4 | 低频 | 参数读取 |
| Boot/Upgrade | H7 -> G4 | TBD | 是否需要远程升级待定 |

## 6. 建议 ID 分配框架

具体 ID 数值暂不冻结。

| 范围 | 用途 |
|---|---|
| `0x000-0x07F` | 安全/急停/Fault 高优先级 |
| `0x080-0x0FF` | H7 心跳和全局命令 |
| `0x100-0x1FF` | 前轮驱动命令/反馈 |
| `0x200-0x2FF` | 后轮驱动命令/反馈 |
| `0x300-0x3FF` | 转向驱动命令/反馈 |
| `0x600-0x6FF` | 配置/诊断 |
| `0x700-0x7FF` | Boot/升级/保留 |

注意：

- 如果使用 11-bit 标准 ID，范围如上。
- 如果使用 29-bit 扩展 ID，可编码节点、消息类型、通道、优先级。
- 最终选择取决于固件复杂度和是否需要扩展节点。

## 7. 命令对象

### 7.1 轮毂命令

每个轮毂需要支持：

- 目标速度。
- 目标力矩/电流。
- 使能/禁用。
- 电子刹车命令。
- 控制模式选择：六步、速度 FOC、力矩 FOC、测试模式。

待定：

- 单位。
- 缩放系数。
- 正方向定义。
- 最大变化率。

### 7.2 转向命令

每个转向通道需要支持：

- 目标轮角。
- 目标角速度限制。
- 目标电流/力矩限制。
- 使能/禁用。
- 零位标定命令。

待定：

- 角度单位。
- 零位定义。
- 前后转向同向/反向定义。
- 超限处理。

## 8. 反馈对象

### 8.1 轮毂反馈

- 实际速度。
- 实际相电流/母线电流。
- 母线电压。
- MOS/板温。
- Hall 状态。
- 控制模式。
- Fault 位。

### 8.2 转向反馈

- 输出轴实际角度。
- 电机反馈速度。
- 相电流/估算力矩。
- 限位状态。
- 编码器状态。
- 堵转状态。
- Fault 位。

## 9. Fault 位建议

| 位 | 名称 | 说明 |
|---:|---|---|
| 0 | `FAULT_OVERCURRENT` | 过流 |
| 1 | `FAULT_OVERVOLTAGE` | 过压 |
| 2 | `FAULT_UNDERVOLTAGE` | 欠压 |
| 3 | `FAULT_OVERTEMP` | 过温 |
| 4 | `FAULT_HALL` | Hall 异常 |
| 5 | `FAULT_ENCODER` | 编码器异常 |
| 6 | `FAULT_CAN_TIMEOUT` | CAN 超时 |
| 7 | `FAULT_ESTOP` | 急停 |
| 8 | `FAULT_DRIVER` | Gate Driver Fault |
| 9 | `FAULT_STALL` | 堵转 |
| 10 | `FAULT_LIMIT` | 限位异常 |
| 11 | `FAULT_CONFIG` | 参数错误 |

## 10. 状态机建议

| 状态 | 含义 |
|---|---|
| `INIT` | 初始化 |
| `SELF_TEST` | 自检 |
| `STANDBY` | 待机，驱动不使能 |
| `ARMED` | 已准备，等待运行命令 |
| `RUNNING` | 运行 |
| `STOPPING` | 受控停车 |
| `FAULT` | 故障锁存 |
| `ESTOP` | 急停锁存 |

## 11. 待用户确认

1. CAN FD 仲裁段和数据段速率。
2. 使用 11-bit 还是 29-bit ID。
3. 上位机是否也接入内部电机 CAN。
4. CAN 超时后是自由滑行、主动刹车、短三相，还是缓停。
5. 是否要求 H7 通过 CAN 给 G4 升级固件。
6. 是否需要外部工具直接诊断 G4 节点。
