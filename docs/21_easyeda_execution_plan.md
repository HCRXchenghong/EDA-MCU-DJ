# EasyEDA 执行计划

## 1. 当前状态

当前还没有 EasyEDA 工程文件。

不建议立即完整开画功率原理图，因为以下参数未定：

- 电池最高电压。
- 最大持续/峰值电流。
- MOS/Gate Driver 选型。
- H7/G4 具体型号和封装。
- 板框、铜厚、散热。

可以先在 EasyEDA 中做：

- 项目创建。
- 多页原理图结构。
- 系统框图页。
- 网名规范。
- 连接器占位。
- MCU 资源占位页。

## 3. 可以先画的内容

### 3.1 System_Block

可以直接画：

- H7。
- 三个 G4。
- ESP32。
- 4 个轮毂。
- 2 个转向。
- 仓门。
- 电子锁。
- 电源树框图。
- CAN/UART/USB Type-C/Ethernet 网口接口框图。

### 3.2 Safety_Chain 框架

可以先画：

- 急停输入端子占位。
- 硬件锁存逻辑占位。
- 驱动 Enable 输出占位。
- 接触器控制占位。
- H7 状态输入占位。

不能定：

- 急停输入常开/常闭。
- 复位逻辑。
- 接触器驱动参数。

### 3.3 Connectors_Testpoints 占位

可以先画：

- 连接器功能框。
- 针脚数量占位。
- 测试点清单。

不能定：

- 连接器封装。
- 大电流针脚尺寸。
- 板边位置。

## 4. 必须等参数再画的内容

| 内容 | 等待参数 |
|---|---|
| MOSFET 选型 | 电流、散热、耐压 |
| 采样电阻 | 峰值/RMS 电流、ADC 范围 |
| 过流阈值 | 保护策略 |
| DCDC 型号 | 电压、电流、散热 |
| 主保险 | 总电流 |
| 预充电阻 | 母线电容、预充时间 |
| 接触器驱动 | 线圈电压/电流 |
| PCB 线宽 | 铜厚、温升、电流 |
| ESP32 天线 | 外壳/天线方案 |

## 5. 推荐执行顺序

1. 在 EasyEDA 建项目和 14 个页面。
2. 画系统框图页。
3. 建立全局网名和页端口。
4. 画 H7/G4/ESP32 占位，不锁定封装。
5. 根据用户参数确定 MCU 型号和封装。
6. 画 H7 最小系统。
7. 画 G4 最小系统模板。
8. 画通信物理层。
9. 画安全链路。
10. 等功率参数确认后画电源和驱动功率页。
11. ERC。
12. 原理图审查。
13. 导入 PCB。

## 6. 使用 EasyEDA API 前提

如果要让我直接操作 EasyEDA，需要：

1. 启动 EasyEDA 桌面客户端。
2. 安装并加载 API 网关扩展。
3. 让我启动/连接 EasyEDA Bridge。
4. 当前只连接一个 EasyEDA 窗口，或你指定窗口。

## 7. 第一轮 EasyEDA 交付物

不需要等所有电气参数即可交付：

- 空项目。
- 多页原理图结构。
- 系统框图。
- 页面接口端口。
- TODO 标记。
- 文档链接。

但第一轮不能用于打样。

## 8. 第二轮 EasyEDA 交付物

需要用户补齐 P0 参数后：

- H7/G4 具体器件。
- 完整最小系统。
- 电源树。
- 通信接口。
- 安全链路。
- 驱动页模板。
- 连接器初版。

## 9. 第三轮 EasyEDA 交付物

需要电机/电流/连接器/散热全部确认：

- 完整功率驱动。
- 完整采样。
- 完整保护。
- 完整 BOM。
- 原理图 ERC。
- PCB 初版布局。
