# 固件需求初版

## 1. 固件分层

| 层 | 模块 | 说明 |
|---|---|---|
| 应用层 | 整车状态机 | H7 主控 |
| 调度层 | 任务调度/通信调度 | H7 |
| 安全层 | 急停、Fault、接触器、超时 | H7 + G4，本地硬件优先 |
| 驱动层 | CAN、UART、SPI、USB Type-C、ETH、ADC、PWM | H7/G4 |
| 电机控制层 | 六步、FOC、位置/速度/电流环 | G4 为主，仓门在 H7 |
| 诊断层 | 日志、参数、错误码 | H7 主导 |

## 2. H7 固件职责

### 2.1 必须实现

- 上电自检。
- 电源轨状态检测。
- 急停状态读取。
- 主接触器/预充控制。
- 与 3 个 G4 的 CAN FD 通信。
- 与 ESP32 的 UART/SPI 通信。
- 外部上位机通信：USB Type-C、Ethernet 网口、CAN/TBD。
- 8 路超声波数据采集。
- 4 路扩展 UART 管理。
- RS485 通信。
- IMU 采集。
- 仓门电机控制。
- 电子锁控制。
- 故障锁存和诊断。
- 参数保存。

### 2.2 H7 状态机

| 状态 | 进入条件 | 动作 | 退出条件 |
|---|---|---|---|
| `BOOT` | 上电/复位 | 初始化时钟、外设、IO 默认态 | 自检开始 |
| `SELF_TEST` | BOOT 完成 | 读电源、急停、通信节点 | 自检通过 |
| `PRECHARGE` | 主电请求且无故障 | 控制预充 | 母线电压达标 |
| `STANDBY` | 电源正常 | 通信运行，驱动禁用 | 上位机使能 |
| `RUN_READY` | 上位机授权 | 下发驱动准备命令 | 所有节点 ready |
| `RUNNING` | 运行命令 | 周期下发速度/角度/锁/门命令 | 停车/故障 |
| `STOPPING` | 停车命令或断联策略 | 受控降速 | 停稳 |
| `FAULT` | 任意故障 | 禁用驱动，记录故障 | 人工/软件复位，规则 TBD |
| `ESTOP` | 急停触发 | 硬件关断，记录急停 | 人工复位 TBD |

## 3. G4 固件职责

### 3.1 前轮/后轮 G4

每个节点控制两个轮毂。

必须实现：

- 上电默认 PWM 关闭。
- CAN 心跳监测。
- Hall 六步控制。
- 有感 FOC。
- 速度闭环。
- 电流/力矩闭环。
- 电子刹车。
- 过流、过压、欠压、过温保护。
- Hall 异常保护。
- 编码器异常保护，若安装编码器。
- CAN 超时停车。
- Fault 锁存。
- 参数读取/保存。

### 3.2 转向 G4

必须实现：

- 输出轴 SPI 绝对编码器读取。
- 输出轴角度位置闭环。
- 双路转向控制。
- 机械限位输入读取。
- 软件限位。
- 堵转保护。
- CAN 心跳监测。
- 急停/Fault 硬件关断响应。
- 零位标定流程。

## 4. 仓门固件

由 H7 负责。

必须实现：

- 开门命令。
- 关门命令。
- 停止命令。
- 限位检测。
- 编码器读取。
- 电流/堵转保护。
- 超时保护。
- 防夹输入，若存在。
- 上电默认不动作。

待确认：

- 仓门电机控制方式：FOC、六步、PWM 驱动、现成驱动器接口。
- 仓门是否要位置闭环。

## 5. 电子锁固件

必须实现：

- 开锁/上锁动作。
- 脉冲宽度或保持时间控制。
- 锁状态读取。
- 过流/短路保护。
- 动作超时判断。
- 手动应急状态记录，若有检测。

待确认：

- 锁是脉冲动作还是持续保持。
- LOCK_STATUS 类型。

## 6. 通信超时策略

| 链路 | 超时动作 | 当前状态 |
|---|---|---|
| 上位机 -> H7 | 受控停车或进入 FAULT | TBD_BY_USER |
| H7 -> 前轮 G4 | 前轮进入安全停车 | TBD_BY_USER |
| H7 -> 后轮 G4 | 后轮进入安全停车 | TBD_BY_USER |
| H7 -> 转向 G4 | 转向保持/回中/禁用 TBD | TBD_BY_USER |
| H7 -> ESP32 | 不影响安全执行器 | 默认可继续运行 |

## 8. 日志与诊断

建议 H7 至少记录：

- 上电次数。
- 最近 N 次故障码。
- 急停触发次数。
- CAN 超时次数。
- 电池欠压/过压次数。
- 电机过流次数。
- 最高温度。
- 固件版本。
- 参数版本。

是否需要外部存储由用户确认。

## 9. 固件开发阶段

| 阶段 | H7 | G4 |
|---|---|---|
| F0 | 板级启动、GPIO 默认态 | 板级启动、PWM 默认关 |
| F1 | CAN/USB Type-C/UART/Ethernet 基础通信 | CAN、Hall、ADC、PWM 基础 |
| F2 | 安全状态机 | 六步换相 |
| F3 | 传感器与外设 | 单路 FOC |
| F4 | 仓门/电锁 | 双路 FOC |
| F5 | 上位机协议 | 转向位置闭环 |
| F6 | 诊断/参数/升级 | 参数和保护完善 |

## 10. 不确定项

以下不能靠猜：

- 电机极对数。
- Hall 相序。
- 编码器分辨率和协议。
- 电流采样比例。
- 电流保护阈值。
- 急停复位策略。
- 上位机协议。
- CAN 速率。
