# BOM 模板

## 1. BOM 字段

正式 BOM 建议至少包含：

| 字段 | 说明 |
|---|---|
| `Item` | 序号 |
| `Reference` | 位号 |
| `Quantity` | 数量 |
| `Category` | 类别 |
| `Value` | 参数值 |
| `MPN` | 原厂料号 |
| `Manufacturer` | 原厂 |
| `Package` | 封装 |
| `Voltage_Rating` | 耐压 |
| `Current_Rating` | 电流 |
| `Power_Rating` | 功率 |
| `Tolerance` | 精度 |
| `Temp_Range` | 温度范围 |
| `Lifecycle` | 生命周期/是否量产 |
| `Supplier` | 供应商 |
| `LCSC_Part` | 嘉立创/LCSC 编号，若使用 |
| `Substitute_1` | 替代料 1 |
| `Substitute_2` | 替代料 2 |
| `DNP` | 是否不贴 |
| `Safety_Critical` | 是否安全关键 |
| `Notes` | 备注 |

## 3. DNP/可选项建议

可做成 DNP 或跳线配置：

- CAN 120R 终端。
- UART 3.3 V/5 V 选择。
- V_DOOR 独立输入/板上供电选择。
- V_LOCK 独立输入/板上供电选择。
- SPI 编码器可选接口。
- ABI 编码器预留接口。
- 外部 CAN 隔离与非隔离版本，若成本压力大。
- Ethernet 已确认为必须项，不作为首版 DNP 裁剪项。

## 4. 不允许没有替代料的关键器件

建议至少准备替代料：

- MOSFET。
- Gate Driver。
- DCDC。
- CAN FD 收发器。
- RS485 收发器。
- TVS/ESD。
- 大电流连接器。
- 采样电阻。

## 5. BOM 审查规则

每个关键器件必须审：

1. 额定电压是否足够。
2. 连续电流和脉冲电流是否足够。
3. 功耗和温升是否可接受。
4. 封装是否可制造。
5. 是否有库存。
6. 是否有替代料。
7. 是否需要散热片/铜皮/过孔。
8. 是否与控制电平兼容。
9. 是否能通过嘉立创或目标供应链采购。
