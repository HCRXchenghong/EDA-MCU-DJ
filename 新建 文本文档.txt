# 外卖小车底盘控制大板最新定版

## 1. 总体架构

采用 一块大 PCB，内部模块化分区：

- STM32H7 主控区
- 前轮双 FOC 驱动区
- 后轮双 FOC 驱动区
- 双转向 FOC 驱动区
- H7 直控仓门无刷编码电机区
- H7 直控电子锁区
- ESP32-S3 板载无线通信区
- CAN FD / UART / RS485 / USB / Ethernet 通信区
- 电源安全区
- 扩展接口区

原则：

- 上位机只和 STM32H7 通信
- ESP32 只服务 H7，不直接控制执行器
- 仓门、电锁、急停、安全逻辑都归 H7
- 不做 7 个电机 MCU
- 不做单 MCU 控全部底盘电机

---

## 2. MCU / 模块分工

| 区域 | MCU / 模块 | 功能 |
|---|---|---|
| 主控区 | STM32H7 | 上位机通信、传感器、调度、安全、仓门、电锁 |
| 前轮驱动区 | STM32G4 | 前左轮 + 前右轮 FOC |
| 后轮驱动区 | STM32G4 | 后左轮 + 后右轮 FOC |
| 转向驱动区 | STM32G4 | 前转向 + 后转向 FOC |
| 无线区 | ESP32-S3 模块 | Wi-Fi AP + 蓝牙，只做 H7 无线通道 |
| UART 扩展 | SPI-UART 扩展芯片 | 8 路超声波 UART |

---

## 3. 电机分配

| 电机 | 数量 | 控制方式 |
|---|---:|---|
| 前轮轮毂 | 2 | 一个 STM32G4 双路 FOC |
| 后轮轮毂 | 2 | 一个 STM32G4 双路 FOC |
| 转向无刷 | 2 | 一个 STM32G4 双路位置 FOC |
| 仓门无刷编码电机 | 1 | STM32H7 直接控制 |

轮毂参数：

- 24–48V
- 500W
- 三相线：U / V / W
- 6 根霍尔线：5V、GND、Hall A、Hall B、Hall C、AUX/Temp/Speed
- 支持电子刹车

---

## 4. 轮毂 FOC 驱动区

前轮一组，后轮一组。

每组包含：

- STM32G4
- 两套三相 MOS 半桥
- 两套 Gate Driver
- 两套电流采样
- 两套 Hall 输入
- 两套编码器预留接口
- 母线电压采样
- 温度采样
- CAN FD
- SWD
- Fault
- Driver Enable
- 急停硬件关断

控制策略：

- 硬件按 FOC 标准设计
- 固件第一阶段：Hall 六步保底
- 固件第二阶段：有感 FOC
- 支持速度控制、力矩/电流控制、电子刹车、CAN 超时停车

关键要求：

- 电流采样一开始就按 FOC 做
- 推荐三低边采样或三相采样
- 不建议只做单母线电流采样

---

## 5. 转向 FOC 驱动区

两个转向电机共用一个 STM32G4。

必须支持：

- 双路 FOC 位置控制
- 前转向输出轴绝对编码器
- 后转向输出轴绝对编码器
- Hall / 电机反馈
- 软件限位
- 机械限位输入
- 堵转保护
- CAN 超时保护
- 急停关断

重点：

转向角必须测输出轴，不能只测电机轴。

---

## 6. 转向输出轴编码器接口

前后转向各预留 1 路输出轴绝对编码器接口：

- J_STEER_ANGLE_F
- J_STEER_ANGLE_R

推荐使用 SPI 绝对磁编码器，接入转向 STM32G4。

每路接口：

- 3.3V_ENC
- GND
- SPI_SCK
- SPI_MISO
- SPI_MOSI，可选
- SPI_CS
- DRDY / ERR，可选
- SHIELD / GND

硬件要求：

- SCK / MISO / MOSI / CS 串 22Ω–100Ω
- 加 ESD
- 3.3V 供电滤波
- 编码器线束尽量短
- 不和三相电机线并走
- 编码器安装在转向输出轴端部
- 输出轴同轴安装磁铁
- 编码器小板固定、防松、防偏心、防水

用途：

- 转向 G4 用它做真实轮角闭环
- H7 只下发目标转向角

---

## 7. 仓门电机区

仓门为 5–24V 无刷编码电机，由 STM32H7 直接控制。

仓门电源：

- V_DOOR = 5–24V
- 建议预留独立 V_DOOR 输入
- 可选从板上 24V 供电
- 仓门 MOS / Driver 建议按 40V 或 60V 器件选型

仓门驱动包含：

- 一套三相 MOS 半桥
- 一套 Gate Driver
- 电流采样
- V_DOOR 母线采样
- ABI 或 SPI 编码器接口
- 开门限位
- 关门限位
- 可选防夹输入
- 过流保护
- 堵转保护
- 超时保护

---

## 8. 电子锁区

电子锁为 5–24V，由 STM32H7 控制。

电子锁电源：

- V_LOCK = 5–24V
- 建议预留独立 V_LOCK 输入
- 可选从板上 24V 供电

电子锁接口：

- V_LOCK+
- LOCK_OUT-
- LOCK_STATUS
- GND

电子锁驱动：

- H7 GPIO 控制
- MOSFET 低边开关或继电器驱动
- MOSFET 耐压建议 ≥40V，优先 60V
- 续流二极管 / TVS
- 锁状态检测
- 过流保护或限流
- 手动应急开锁预留

ESP32 不直接控制电子锁。

---

## 9. 编码器预留

每个轮毂建议预留：

- Hall 接口
- ABI 增量编码器接口
- 可选 SPI 磁编码器接口

每个转向必须预留：

- Hall / 电机反馈
- SPI 输出轴绝对磁编码器接口

仓门预留：

- ABI 编码器接口
- 或 SPI 编码器接口

ABI 接口：

- VCC，可跳 3.3V / 5V
- GND
- A
- B
- Z
- Shield / GND

SPI 编码器接口：

- 3.3V
- GND
- SCK
- MISO
- MOSI
- CS
- DRDY / ERR 可选
- Shield / GND

所有编码器信号加：

- 串联电阻
- ESD
- 上拉预留
- 必要时电平转换

---

## 10. CAN FD

全板按 CAN FD 设计，不用 TJA1050。

| CAN | 用途 | 收发器 |
|---|---|---|
| CAN1 | 内部电机 CAN FD | TJA1044 / TCAN1042 类 CAN FD |
| CAN2 | 外部扩展 CAN FD | 建议 ISO1042 隔离 CAN FD |

CAN 要求：

- 支持 CAN FD
- 预留 120Ω 终端电阻跳帽
- CANH / CANL 加 TVS
- 外部 CAN 加共模电感
- 外部 CAN 建议隔离
- 每个驱动区独立 CAN ID
- CAN 掉线后电机自动进入安全状态

---

## 11. UART

总需求：

- 8 路 UART 超声波
- 4 路扩展 UART
- 1 路 RS485
- 1 路 ESP32 UART
- 1 路调试 UART 或 USB CDC

分配：

| UART 类型 | 实现 |
|---|---|
| 8 路超声波 | SPI 转多路 UART 扩展芯片 |
| 4 路扩展 UART | 优先 STM32H7 原生 UART |
| RS485 | H7 原生 UART + RS485 收发器 |
| ESP32 | H7 原生 UART，必要时可改 SPI |
| 调试 | USB CDC 为主，UART 预留 |

UART 不默认高速，每路波特率软件可配置。

---

## 12. UART 3.3V / 5V 跳线

所有外部 UART 支持跳线帽选择 3.3V / 5V。

每路接口：

- VOUT
- GND
- TX
- RX

跳线逻辑：

| 跳线 | 外设供电 | UART 信号 |
|---|---:|---:|
| 3.3V | 3.3V | 3.3V |
| 5V | 5V | 5V |

关键要求：

- H7 / UART 扩展芯片侧固定 3.3V
- 外设侧由 VSEL 选择 3.3V 或 5V
- TX/RX 必须经过双电源电平转换器
- 不允许 5V UART 直接进入 H7
- 每路加串联电阻、ESD、限流保护

---

## 13. ESP32-S3

ESP32-S3 内嵌在主板上，只作为 H7 的无线通信模块。

负责：

- Wi-Fi AP
- 蓝牙
- 无线调试
- 无线参数配置
- 无线日志通道

不负责：

- 电机控制
- 仓门控制
- 电子锁控制
- 急停
- 安全逻辑

ESP32 硬件要求：

- 3.3V 独立滤波供电
- EN / BOOT
- UART_TX / UART_RX 接 H7
- 可选 SPI 接 H7
- 下载 / 调试测试点
- 天线净空区
- 远离 MOS 功率区和大电流母线

---

## 14. 其他接口

STM32H7 负责：

- USB，上位机 / 调试 / 升级
- Ethernet，交换机 / 工控机 / ROS 上位机
- IMU，优先 SPI
- RS485，1 路
- CAN FD，2 路
- PWM，2 路
- SPI，2 路
- GPIO 扩展排针
- 3.3V / 5V 输出

扩展接口要求：

- SPI 默认 3.3V 逻辑
- PWM 默认 3.3V，可预留 5V 电平转换
- GPIO 默认 3.3V
- 3.3V / 5V 输出必须限流
- 所有外部 IO 加 ESD 和串联保护电阻

---

## 15. 电源系统

主电源：

- 48V 电池输入
- 支持 24–48V 系统
- 功率器件建议按 75V / 100V 余量设计

电源树：

- 48V 主母线
- 48V → 24V，给 V_DOOR / V_LOCK / 外设
- 48V → 12V / 15V，给 Gate Driver
- 12V / 15V → 5V
- 5V → 3.3V

特殊电源：

- V_DOOR = 5–24V，仓门电机电源
- V_LOCK = 5–24V，电子锁电源

电源安全区包含：

- 总保险
- 反接保护策略
- 预充电路
- 主接触器控制
- 母线电压检测
- 母线电容
- TVS
- 刹车泄放电阻预留
- 5V / 3.3V 外设限流输出

---

## 16. 急停与安全

急停不能只进 MCU，必须硬件关断。

急停链路：

- 急停输入 1
- 急停输入 2
- 输入 H7 做状态判断
- 同时拉低所有驱动区 Enable
- 同时控制主接触器断开
- 所有驱动区急停后关闭 PWM

驱动器必须支持：

- 过流保护
- 过压保护
- 欠压保护
- 过温保护
- Hall 异常保护
- 编码器异常保护
- CAN 超时保护
- Fault 锁存
- 上电默认不使能

---

## 17. PCB 分区

分区：

- 低压主控区：H7、USB、网口、ESP32、IMU、扩展 IO
- 传感器接口区：UART、RS485、超声波接口
- 电源安全区：48V 输入、预充、接触器、电源转换
- 前轮功率区：前左、前右三相桥
- 后轮功率区：后左、后右三相桥
- 转向功率区：前转向、后转向三相桥
- 仓门功率区：V_DOOR 三相桥
- 电子锁区：V_LOCK 锁驱动
- 刹车泄放区：靠近主母线

布局原则：

- 主控区远离 MOS 功率区
- ESP32 远离功率区，天线区域禁铺铜
- 功率区靠板边，方便接电机线和散热
- 每个功率区本地放母线电容
- 大电流回路最短
- 电流采样走线必须干净
- Hall / 编码器 / UART 不穿过大电流区域
- 每个 MCU 预留 SWD
- 每个电源轨预留测试点
- 每个驱动区预留 Fault / PWM / 电流采样测试点

---

## 18. 最终定版

最终方案为：

一块外卖小车底盘控制大板，内部包含 STM32H7 主控区、前轮双 FOC 驱动区、后轮双 FOC 驱动区、双转向 FOC 驱动区、H7 直控 5–24V 仓门无刷编码电机区、H7 直控 5–24V 电子锁区、ESP32-S3 板载无线通信区、CAN FD 通信区、UART 传感器区、转向输出轴绝对编码器接口、USB/网口上位机接口区、电源安全区、编码器预留接口和急停硬件关断系统。
